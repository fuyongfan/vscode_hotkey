# windows 快捷键
## 全局
```
ctrl + shift + p    // 显示命令面板
Ctrl + p            // 快速打开文件
```

## 基本
```
alt + up             // 向上移动
alt + down           // 向下移动
shift + alt + up     // 向上复制一行
shif + alt + down    // 向下复制一行
ctrl + shift + k     // 删除行
ctrl + shift + enter // 向上插入一行
ctrl + enter         // 向下插入一行
ctrl + [             // 减少缩进
ctrl + ]             // 增加缩进
home                 // 行首
ctrl + home          // 文件开头
end                  // 行尾
ctrl + end           // 文件结尾
ctrl + up            // 按行向上滚动
ctrl + down          // 按行向下滚动
ctrl + pgUp          // 按屏向上滚动
ctrl + pgDown        // 按屏向下滚动
```
### 折叠
```
ctrl + shift + [     // 折叠代码块
ctrl + shift + ]     // 展开代码块
ctrl + k ctrl + [    // 折叠全部子代码块
ctrl + k ctrl + ]    // 展开全部子代码块
ctrl + k ctrl + 0    // 折叠文件全部代码块
ctrl + k ctrl + j    // 展开文件全部代码块
```

### 注释
```
Ctrl + /             // 添加、移除行注释
Shift + Alt + A      // 添加、移除块注释
Alt + Z              // 自动换行、取消自动换行
```
